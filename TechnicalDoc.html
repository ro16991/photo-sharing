Photo Sharing via cloud The purpose of this project is to create a website that allows users to upload and view pictures that they have stored onto a cloud server Both the website code and the images are stored on the same cloud server. A user can register a new account or log in with already existign credentials, while logged in they can either view imagee they have already uploaded and stored on the cloud from wherever or they can upload new ones. They can also view other peoples images. Getting Started (Software Based) Prerequisites As this is a website, the user would only need a username and password to log on. Admin users would need an admin username and password. Installing Ensure you have a stable internect connection. Load the link to the website homepage. You can then sign up with your details or if you already have an account you can sign in. Running Tests Ensure the link to the webpage runs from any device. User should be able to view images on the site that are not stored on their computers. They can upload or view their own stored images.
